# OpenCode 懶人包 #06：生圖技能

> 版本：v0.5
> 更新日期：2026-07-17

## 這個懶人包會幫你做什麼？

安裝可直接使用的 `opencode-draw` Skill，透過 OpenAI `gpt-image-2` 產生 PNG。Skill 內已包含 `draw.py`，不會再下載或借用其他代理工具的生圖 Skill。

## 步驟一：安全設定 API Key

可使用系統環境變數，或在使用者家目錄建立 `~/.openai.env`：

```text
OPENAI_API_KEY=<你的金鑰>
```

安全要求：

- 不可使用 `type`、`cat`、`Get-Content` 顯示整個檔案
- 不在對話、截圖或完成回報中顯示金鑰
- 專案 `.gitignore` 必須忽略 `.openai.env`
- 只回報「已設定／未設定」

Windows PowerShell 可只檢查是否存在：

```powershell
if ($env:OPENAI_API_KEY) { "OPENAI_API_KEY 已設定" } else { "OPENAI_API_KEY 未設定；程式也會檢查 ~/.openai.env" }
```

## 步驟二：安裝 Skill

從完整懶人包 repo 根目錄執行同步腳本：

```powershell
powershell -ExecutionPolicy Bypass -File "skills/07-install-all/install-opencode-skills.ps1" -SourceRoot "skills" -Force
```

安裝後應同時存在：

```text
~/.config/opencode/skills/opencode-draw/SKILL.md
~/.config/opencode/skills/opencode-draw/draw.py
```

`opencode-draw` 只能執行同目錄的 `draw.py`。不得搜尋、載入、複製或改用 `~/.codex/skills/codex-draw`；若 `draw.py` 缺失，停止生圖並從本 repo 重新同步，不得刪除或簡化 `opencode-draw`。

## 步驟三：測試

### Windows PowerShell

```powershell
uv run --with openai python "$HOME/.config/opencode/skills/opencode-draw/draw.py" "一隻橘貓坐在窗邊看夕陽，水彩風格" --name test-cat --quality low
```

### macOS / Linux

```bash
uv run --with openai python ~/.config/opencode/skills/opencode-draw/draw.py "一隻橘貓坐在窗邊看夕陽，水彩風格" --name test-cat --quality low
```

實際輸出位置以程式回報為準：有 `slides/` 時是 `slides/generated/`，否則是 `generated/`。
