把使用 OpenCode 懶人包所需的環境一次建置完成：

## 請 OpenCode 幫我執行以下步驟

> 把這份檔案的內容貼給 OpenCode，它會自動執行。

---

### 步驟一：環境檢查

請 OpenCode 先確認以下所有項目：

1. 作業系統：Windows / macOS / Linux
2. 網路連線
3. **Node.js**：`node --version`（需 18+）
4. **uv**（Python 工具管理器）：`uv --version`

把缺少的記錄下來，後續步驟逐一安裝。

---

### 步驟二：安裝 Node.js（如果未安裝或版本 < 18）

**Windows**：
```bash
winget install --id OpenJS.NodeJS --accept-source-agreements --accept-package-agreements
```

**macOS**：
```bash
brew install node
```

**Linux（Ubuntu/Debian）**：
```bash
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt install -y nodejs
```

確認：`node --version`

---

### 步驟三：安裝 OpenCode

```bash
npm install -g opencode-ai
```

確認版本：
```bash
opencode --version
```

---

### 步驟四：安裝 uv（如果未安裝）

**Windows（PowerShell）**：
```bash
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
```

**macOS / Linux**：
```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

確認：`uv --version`

---

### 步驟五：最終驗證

```bash
node --version
opencode --version
uv --version
```

> ✅「環境建置完成！你的電腦已經準備好使用所有 OpenCode 懶人包了。」

## 常見問題

| 問題 | 解法 |
|------|------|
| `opencode` 指令找不到 | 確認 npm global bin 在 PATH 中：`npm config get prefix`，把 `prefix/bin` 加入 PATH |
| winget 找不到 | Windows 10 1809+ 內建；或到 Microsoft Store 安裝「App Installer」 |
| Node.js 版本太舊 | 移除舊版重裝最新 LTS |
