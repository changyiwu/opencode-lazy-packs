# OpenCode 懶人包 #05：安裝瀏覽器控制

> 版本：v0.4
> 更新日期：2026-07-17

## 這個懶人包會幫你做什麼？

- Playwright MCP：網頁導航、擷取與互動
- open-computer-use：桌面與應用程式控制

兩者權限很高，安裝後仍應在登入、送出表單、購買、刪除或發布前取得確認。

## 步驟一：Playwright MCP

可使用 `opencode mcp add`，或安全合併以下設定：

```json
{
  "$schema": "https://opencode.ai/config.json",
  "mcp": {
    "playwright": {
      "type": "local",
      "command": ["npx", "-y", "@playwright/mcp"],
      "enabled": true
    }
  }
}
```

## 步驟二：open-computer-use

```bash
npm install -g open-computer-use
open-computer-use --version
open-computer-use install-opencode-mcp
```

若自動設定失敗，再手動加入：

```json
{
  "mcp": {
    "open-computer-use": {
      "type": "local",
      "command": ["open-computer-use", "mcp"],
      "enabled": true
    }
  }
}
```

## 步驟三：驗證

```bash
opencode mcp list
```

依序測試：

1. Playwright：開啟 `https://example.com` 並讀取標題。
2. open-computer-use：在使用者確認後截取桌面或指定視窗。

## 輕量替代方案

若只要簡單擷取，先用 `cli-hub search browser` 確認目前可用套件，再決定是否安裝；不要假設 registry 中永遠存在固定名稱。

```bash
pip install cli-anything-hub
cli-hub search browser
```
